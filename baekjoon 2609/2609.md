## 최종 답안

```python

# 두 정수를 입력받는다
a, b = input().split()
a = int(a)
b = int(b)

# 최대 공약수를 구하는 함수
def GCD(i, j):
    if j == 0:
        return i
    else:
        return GCD(j, i % j)


gcd = GCD(a, b)
lcm = int(a*b/gcd)
print(gcd, lcm)

```

## 유클리드 호제법

> 임의의 자연수 a, b가 있다고 가정해보자.  
> a를 b로 나눈 나머지를 n이라고 했을 때, n == 0이 되면 b는 두 자연수의 최대 공약수이다.  
>
> 만약 n이 0이 아니라면, a에 b 값을 넣고 b값에 n을 넣어 나머지가 0이 될 때까지 반복한다.  

문돌이인 내가 이런 공식을 알리가 있나...   
찾아보니 이런 공식이 있어서 활용하여 문제를 풀었다.  

최소 공배수는 자연수 a,b의 곱을 최대 공약수로 나누면 구할 수 있다  


## input().split()
파이썬 문법을 초짜인 나는 input()을 활용해 값을 하나씩 입력받는 법밖에 몰랐는데,   
split()을 사용하면 공백을 기준으로 여러개의 값을 받을수 있다는 사실을 알게 되었다.   
split()안에 문자를 넣어주면 그 문자를 기준으로 여러개의 값을 받을 수도 있다.  
이를테면  
```python
split(',')
# 이 경우에는 , 를 기준으로 두개 이상의 값을 입력받을 수 있다.
```
