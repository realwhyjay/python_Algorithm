## 답안

```python
N, M = map(int, input().split())
card = list(map(int, input().split()))
sum = 0
max = 0

# 입력받은 리스트 card에 대해 3중 for문을 만들어준다.
for i in card:
    for j in card:
        for t in card:
          # 하나라도 같은 값이 존재하면 continue 해준다.
            if i == j or i == t or j == t:
                continue
            else:
          # 같은 값이 없는 경우에는 sum에 모두 더해주고, sum값을 M값과 최대값인 max와 함께 비교해준다
                sum = i + j + t
                if max < sum <= M:
                    max = sum

print(max)
```

## 모범답안
```python
n, m = list(map(int, input().split(' ')))
data = list(map(int, input().split(' ')))

result = 0
length = len(data)
count = 0

# 나동빈님의 코드와 내 코드가 다른점은 for문의 range를 지정하는 방법이었다.
# 나는 list를 끌어다가 for문을 작성한 반면, 나동빈님은 직접 range를 지정해주었다.
# 이 경우에는 range를 i+1, j+1과 같은식으로 짜서 겹치는 값이 없도록 해주었다.
for i in range(0, length):
    for j in range(i + 1, length):
        for k in range(j + 1, length):
            sum_value = data[i] + data[j] + data[k]
            if sum_value <= m:
              # 나는 직접 매번 최대값과 비교해준것과 달리, max 함수를 사용해서 최댓값을 return 했다.
                result = max(result, sum_value)

print(result)
```
