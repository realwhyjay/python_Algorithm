# 백준 9012 : 괄호

#### Date : 21/06/12

#### Lang : Python

#### Question : [백준 9012](https://www.acmicpc.net/problem/9012)

## 최종 답안 

```python
T = int(input())

for _ in range(T):
    check = 0
    list = input()
    for i in list:
        if i == '(':
            check += 1
        else:
            check -= 1
            if check < 0:
                break

    if check == 0:
        print("YES")
    else:
        print("NO")
```

(의 개수와 )의 개수가 같도록하는게 중요한데, 대신 괄호가 반드시 제대로 닫혀주어야한다는 조건이 있다.

check이라는 변수를 두고 0값을 넣어주었다.  
입력받은 활호들 중에 ( 가 있으면 check 변수를 1씩 늘리고 )는 1씩 빼주었다 .  
만약 check이 음수가 되는 순간이 온다면, 애초에 괄호가 제대로 닫힐 수가 없는 상황이기에 바로 break를 해주어 예외처리를 해주었다.
