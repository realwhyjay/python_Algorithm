## 1차 답안 (틀림)

```python
num = int(input())
f1 = 0
f2 = 1
f3 = 0

for _ in range(num-1):
    f3 = f1 + f2
    f1 = f2
    f2 = f3

print(f3)

```

문제를 너무 단순하게 생각하고 접근했다.   
입력받은 값이 1일 경우 등의 예외 처리를 전혀 해주지 않았다.  

## 1차 답안 수정

```python
num = int(input())
f1 = 0
f2 = 1
f3 = 0

for _ in range(num-1):
    f3 = f1 + f2
    f1 = f2
    f2 = f3

if num == 1:
    f3 = 1

print(f3)

```

1에대한 예외처리를 해주었지만, 생각해보니 변수를 세개씩이나 만들 필요가 없을 것 같아 다시 코드를 짜기로 했다.  

## 최종 답안

```python
num = int(input())
num_list = [0 for i in range(num+1)]
num_list[1] = 1

for i in range(2, num+1):
    num_list[i] = num_list[i-1]+num_list[i-2]

print(num_list[num])

```

리스트를 활용해 보다 간결하고 효율적으로 코드를 짜주었다.


## ndb 답안

```python
num = int(input())

a, b = 0, 1
while num > 0:
    a, b = b, a+b
    num -= 1

print(a)
```

나동빈님 코드를 봤는데 내가 접근한 것과는 완전 다른 방식으로 풀었다.  
훨씬 효율적이고 간단한듯 하다.
