## 1차 답안 (실패)

```javascript
e, s, m = map(int, input().split())
E, S, M = 0, 0, 0
count = 0

while(E != e or S != s or M != m):
    count += 1
    E += 1
    S += 1
    M += 1

    if(E > e):
        E -= e
    if(S > s):
        S -= s
    if(M > m):
        M -= m

print(count)

```

반례
1 2 3 
정답은 5266이지만 위의 코드로는 6이 나온다.

해당 반례가 나오는 이유는 내가 문제를 잘못 이해했기 때문이다.
애초에 문제에는 각 행성의 년수 범위를 각각 15,28,19로 정해주었는데,
나는 입력받는 숫자들이 그 범위의 최대치라고 오해했다 ㅠ

## 2차 답안

```python
e, s, m = map(int, input().split())
E, S, M = 0, 0, 0
count = 0

while(E != e or S != s or M != m):
    count += 1
    E += 1
    S += 1
    M += 1

    if(E == 16):
        E = 1

        S =1

        M =1
print(count)


```



## 모범 답안

e, s, m = map(int, input().split())

count = 1
while True:
