# 백준 15654 : N과 M(5)

#### Date : 21/04/22

#### Lang : Python

#### Question : [백준 15654](https://www.acmicpc.net/problem/15654)

## 최종 답안 

```javascript
from itertools import permutations

n, m = map(int, input().split())
num = list(map(int, input().split()))

result = list(permutations(sorted(num), m))

for k in result:
    for j in k:
        print(j, end=' ')
    print()

```
