## 1차 답안 

```python
m, n = map(int, input().split())

num = [i for i in range(m, n+1)]

for i in range(len(num)):
    for k in range(i+1, len(num)):
        if num[i] == 0:
            continue
        if num[k] % num[i] == 0:
            num[k] = 0

for i in num:
    if i != 0:
        print(i)

```

에라토스테네스의 체를 활용하여, 주어진 범위 내의 소수를 출력하게 해주었다.  
간만에 혼자 힘으로 문제를 풀어 뿌듯했는데

## 모범 답안 

```python
m, n = map(int, input().split())

def isprime(m, n):
  n += 1                            # for문 사용을 위한 n += 1
  prime = [True] * n                # n개의 [True]가 있는 리스트 생성
  for i in range(2, int(n**0.5)+1): # n의 제곱근까지만 검사
    if prime[i]:                    # prime[i]가 True일때
      for j in range(2*i, n, i):    # prime 내 i의 배수들을 False로 변환
        prime[j] = False

  for i in range(m, n):
    if i > 1 and prime[i] == True:  # 1 이상이면서 남은 소수들을 출력
      print(i)

isprime(m, n)
```

모범답안을 찾아보니, 모든 경우의 수를 체크해주는 것이 아니라  
n의 제곱근 만큼의 수만 검사를 해줌으로써 cost를 대폭 줄였다.
