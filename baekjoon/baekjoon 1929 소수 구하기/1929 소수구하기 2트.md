# 백준 1929 : 소수 구하기 2트

#### Date : 21/08/28

#### Lang : Python

#### Question : [백준 1929](https://www.acmicpc.net/problem/1929)

## 최종 답안 

```python
M, N = map(int, input().split())

check = [True]*(N+1)

for i in range(2, int((N+1)**0.5)+1):
    if check[i] == True:
        for j in range(2*i, N+1, i):
            check[j] = False

for k in range(M, N+1):
    if k > 1 and check[k] == True:
        print(k)

```

이전에 풀었을 때도, N의 제곱근 만큼의 수만 검사를 해주는걸 몰랐었는데 이번에도 똑같은 부분에서 막혔다.  
이번에 풀 때는, 제곱근의 값에 +1을 해주지 않아 반례가 생기는 경우가 일어났다.  
소수인지 아닌지를 체크해주는 boolean 배열의 인덱스와, for문들이 겹치면서 헷갈렸던듯하다.
