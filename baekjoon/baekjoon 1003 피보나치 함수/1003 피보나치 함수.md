## 최종 답안 

```python
for _ in range(int(input())):
    cnt0 = [1, 0]
    cnt1 = [0, 1]
    n = int(input())

    for i in range(2, n+1):
        cnt0.append(cnt0[i-1]+cnt0[i-2])
        cnt1.append(cnt1[i-1]+cnt1[i-2])

    print(cnt0[n], cnt1[n])

```

각 n 값 별로 0을 호출한 수와 1을 호출한 수를 체크하여 규칙을 찾고  
점화식을 만들어 문제를 해결한다.  
규칙만 찾으면 금방 풀 수 있을 것 같은데, 이걸 어케 단박에 찾아내는지요..? ㅂㄷㅂㄷ
