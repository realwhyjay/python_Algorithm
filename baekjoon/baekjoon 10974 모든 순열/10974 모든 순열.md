# 백준 10974 : 모든 순열

#### Date : 21/04/07

#### Lang : Python

#### Question : [백준 10974](https://www.acmicpc.net/problem/10974)

## 최종 답안

```javascript
from itertools import permutations

n = int(input())
lst = []

for i in range(1, n+1):
    lst.append(i)

result = list(permutations(lst, n))

for j in result:
    for k in j:
        print(k, end=' ')
    print()
```
