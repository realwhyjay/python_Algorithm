# 백준 10809 : 알파벳 찾기

#### Date : 21/07/20

#### Lang : Python

#### Question : [백준 10809](https://www.acmicpc.net/problem/10809)

## 최종 답안 

```python
string = list(input())
n = [-1]*26

for i in range(len(string)):
    if (n[ord(string[i])-ord('a')] == -1):
        n[ord(string[i])-ord('a')] = i

for i in n:
    print(i, end=' ')

```

후위 표기식 문제를 풀었던 걸 바탕으로 접근하니 쉽게 풀 수 있었다.
