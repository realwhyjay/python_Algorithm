# 백준 1874 : 스택 수열

#### Date : 21/02/09

#### Lang : Python

#### Question : [백준 1874](https://www.acmicpc.net/problem/1874)


## 답안

```python
num = int(input())
stack = []
result = []
count = 1
temp = True # 불가능 여부를 체크하기 위한 변수

# 입력 받은 수만큼 반복한다
for i in range(num):
  # for문이 돌아가는 동안 수열을 이루는 정수를 입력받는다.
    check = int(input())
    # 총 몇개의 수가 들어갔는지를 체크하는 count와 check을 비교해준다.
    # 입력받은 정수 check 만큼의 수가 수열에 들어와있지 않는다면 해당 수를 stack에 push
    # 우리는 최종적으로 그 수열을 이루기까지의 과정을 출력해주어야하기 때문에, 이를 위한 result 리스트에 +를 push 해준다.
    # 해당 작업이 끝났다면 count를 1씩 늘려준다
    while count <= check:
        stack.append(count)
        result.append('+')
        count += 1
    # 수열에 들어온 가장 마지막 원소가 check과 같다면 pop을 해준다.
    # pop했음을 나타내는 -를 result에 push 해준다.
    if stack[-1] == check:
        stack.pop()
        result.append('-')
    # 만약 수열이 완성되지 않는다면 False 처리를 해준다.
    else:
        temp = False

# 결과 출력
if temp == False:
    print("NO")
else:
    for i in result:
        print(i)

```
