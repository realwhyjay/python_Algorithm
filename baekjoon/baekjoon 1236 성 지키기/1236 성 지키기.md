## ndb 클론 코딩

```python
n, m = map(int, input().split())
castle = []

# 성 상태를 입력받는다.
for _ in range(n):
    castle.append(input())

# 행과 열을 체크하기 위해 입력받은 n,m의 수만큼 0으로 채워진 리스트를 만든다.
row = [0] * n
colum = [0] * m

# 입력받은 성의 상태 중, 경비원의 위치를 파악한다.
for i in range(n):
    for j in range(m):
        if castle[i][j] == 'X':
            row[i] = 1
            colum[j] = 1

# 경비원이 없는 열을 체크한다
row_count = 0
for i in range(n):
    if row[i] == 0:
        row_count += 1

# 경비원이 없는 행을 체크한다
colum_count = 0
for i in range(m):
    if colum[i] == 0:
        colum_count += 1

# 경비원이 없는 열과 행의 수 중 더 큰 값을 출력한다.
print(max(row_count, colum_count))

```



## 다른 답안

```python
n,m = map(int,input().split())
castle=[]

for _ in range(n):
    castle.append(input())

row = []
col = []

# castle[i]에 X가 없으면 True를 append한다.
for i in range(n):
    row.append("X" not in castle[i])

# castle[i][j]에 X가 없으면 True를 append한다.
for j in range(m):
    col.append("X" not in [castle[i][j] for i in range(n)])    

# 입력받은 list에 sum 함수를 사용하면 해당 리스트의 모든 원소를 더해준다.
# 그런데 row처럼 bool값을 가진 리스트는 True의 갯수를 리턴해준다.
print(max(sum(row),sum(col)))
```

해당 답안은 나동빈님의 답안처럼 0으로 가득찬 리스트를 만들어두는게 아니라, 
빈 값에 bool 값을 추가해주는 방식을 사용했다.
