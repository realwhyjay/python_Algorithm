#### Question : [백준 10773](https://www.acmicpc.net/problem/10773)

## 최종 답안 

```python
import sys

k = int(input())
money = []
for _ in range(k):
    num = int(sys.stdin.readline())
    if num == 0:
        money.pop()
    else:
        money.append(num)

print(sum(money))

```
