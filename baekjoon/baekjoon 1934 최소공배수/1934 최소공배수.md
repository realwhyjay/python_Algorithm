# 백준 1934 : 최소공배수

#### Lang : Python

#### Question : [백준 1934](https://www.acmicpc.net/problem/1934)

## 최종 답안 

```python
num = int(input())


def GCD(a, b):
    if a < b:
        a, b = b, a
    if b == 0:
        return a
    else:
        return GCD(b, a % b)


while (num != 0):
    num1, num2 = map(int, input().split())
    lcm = int(num1*num2/GCD(num1, num2))
    print(lcm)
    num -= 1

```

최대 공약수와 최소공배수 문제를 바탕으로 쉽게 해결할 수 있었다.
