## 최종 답안 

```javascript
from itertools import combinations_with_replacement

n, m = map(int, input().split())
num = [i for i in range(1, n+1)]

result = list(combinations_with_replacement(sorted(num), m))

for k in sorted(result):
    for j in k:
        print(j, end=' ')
    print()

```

## combinations_with_replacement()

combinations_with_replacement는 기본적으로 combination과 기능이 동일하지만,  
요소의 반복이 가능하다는 점에서 차이점을 가진다.

|                  예시                  |             결과              |
| :------------------------------------: | :---------------------------: |
|         combination('ABCD', 2)         |       AB AC AD BC BD CD       |
| combinaton_with_replacement('ABCD', 2) | AA AB AC AD BB BC BD CC CD DD |
