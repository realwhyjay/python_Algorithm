## 1차 답안 (메모리 초과)

```javascript
import sys

S = []

num = int(input())

for _ in range(num):
    lis = sys.stdin.readline().strip().split()
    if lis[0] == 'add':
        S.append(lis[1])
    elif lis[0] == 'remove':
        if lis[1] not in S:
            continue
        else:
            S.remove(lis[1])
    elif lis[0] == 'check':
        if lis[1] not in S:
            print(0)
        else:
            print(1)
    elif lis[0] == 'toggle':
        if lis[1] not in S:
            S.append(lis[1])
        else:
            S.remove(lis[1])
    elif lis[0] == all:
        S = [i for i in range(1, 21)]
    else:
        S.clear()

```

메모리 초과를 하고 나서보니 list가 아닌 set 함수를 써줘야하는 문제였다..  
아니 문제 제목도 집합인데, 왜 나는 list로 풀으려고 했던거지;

## 최종 답안

```python
import sys

m = int(sys.stdin.readline())
S = set()

for _ in range(m):
    temp = sys.stdin.readline().strip().split()
    
    if len(temp) == 1:
        if temp[0] == "all":
            S = set([i for i in range(1, 21)])
        else:
            S = set()
    
    else:
        func, x = temp[0], temp[1]
        x = int(x)

        if func == "add":
            S.add(x)
        elif func == "remove":
            S.discard(x)
        elif func == "check":
            print(1 if x in S else 0)
        elif func == "toggle":
            if x in S:
                S.discard(x)
            else:
                S.add(x)
```

list S를 set으로 바꿔주고, print를 찍어주는 부분도 파이썬스러운 코드로 바꿔주었다.
