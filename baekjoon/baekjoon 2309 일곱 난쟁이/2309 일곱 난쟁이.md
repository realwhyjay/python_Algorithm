## 1차 답안

```javascript
import sys

dwarf = []
for _ in range(9):
    dwarf.append(int(sys.stdin.readline()))


sum = sum(dwarf)

for i in range(9):
    for q in range(i+1, 9):
        if (sum-(dwarf[i]+dwarf[q]) == 100):
            dwarf.remove(dwarf[i])
            dwarf.remove(dwarf[q])
            break
    if(len(dwarf) != 9):
        break


dwarf.sort()
for i in dwarf:
    print(i)

```

찾은 값의 원소를 remove해주는 과정에서 index에러가 생겼다.
dewarf[i]를 지우고 난 뒤에 dewarf[q]는 dewarf[i]를 지우기 전의 dewarf[q]와는 다른 값을 가지기 때문.

## 최종 답안

```javascript
import sys

dwarf = []
# 입력값을 받는다.
for _ in range(9):
    dwarf.append(int(sys.stdin.readline()))

# 난쟁이 키의 총 합을 미리 계산한다.
sum = sum(dwarf)

for i in range(9):
    for q in range(i+1, 9):
    # 난쟁이 값의 총합에서, 두 난쟁이의 키를 뺐을 때 100일 경우를 찾는다.
        if (sum-(dwarf[i]+dwarf[q]) == 100):
            fake1 = dwarf[i]
            fake2 = dwarf[q]
            dwarf.remove(fake1)
            dwarf.remove(fake2)
            break
    if(len(dwarf) != 9):
        break


dwarf.sort()
for i in dwarf:
    print(i)

```
