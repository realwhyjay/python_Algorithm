## 1차 답안 (클론 코딩)

```javascript
from itertools import permutations

n = int(input())
lst = list(map(int, input().split()))
lst2 = list(permutations(lst, n))
result = []
ans = 0


for a in lst2:
    cnt = 0
    for i in range(n-1):
        cnt += abs(a[i]-a[(i+1)])
    ans = max(cnt, ans)

print(ans)

```

## 순열(persmutations)

순열이란 n개의 원소를 사용해서 순서를 정하여 r개의 배열로 나타내는 것을 말한다. 순열은 순서가 있기 때문에 원소의 종류가 같아도 순서가 다르면 다른배열이 된다.
itertools 모듈의 외장함수로 사용이 가능하다.
