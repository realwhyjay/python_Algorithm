# 백준 1676 : 팩토리 0의 개수

#### Date : 21/06/14

#### Lang : Python

#### Question : [백준 1676](https://www.acmicpc.net/problem/1676)

## 최종 답안

```python
def pactorial(N):
    if N == 0 or N == 1:
        return 1
    return N*pactorial(N-1)


N = int(input())

result = list(str(pactorial(N)))
count = 0

for i in range(len(result)-1, 0, -1):
    if result[i] == "0":
        count += 1
    else:
        break

print(count)

```

팩토리얼을 만드는 문제를 활용하여주었다.  
팩토리얼 값을 문자열로 바꾼뒤 뒤에서부터 조건에 맞는 값을 구하도록 하였다.
