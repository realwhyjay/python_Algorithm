# 백준 1676 : 팩토리얼 0의 개수 2트

#### Date : 21/08/28

#### Lang : Python

#### Question : [백준 1676](https://www.acmicpc.net/problem/1676)

## 최종 답안 

```python
def pactorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * pactorial(n-1)


N = int(input())
result = str(pactorial(N))
answer = 0

for i in range(len(result)-1, -1, -1):
    if (result[i] == "0"):
        answer += 1
    else:
        print(answer)
        break

```

팩토리얼을 구하는 함수를 재활용하였고,  
string 값으로 변환하여, 마지막 요소부터 0값인지 아닌지를 확인해주는 식으로 해결했다.
