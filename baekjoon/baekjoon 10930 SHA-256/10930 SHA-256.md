## 답안

```python
import hashlib

data = input()
# SHA256 메소드를 위해서는 입력받은 문자열을 encode 해주어야 한다고 한다.
encoded_data = data.encode()
# hashlib.sha256(문자열 바이트 객체).hexidigest() : 해시 결과 문자열
result = hashlib.sha256(encoded_data).hexdigest()

print(result)

```

## SHA 256  
SHA256은 64자리 문자열을 반환하는 암호화 방식이라고 한다.  
파이썬에서 SHA256 암호화를 하기 위해서는, 해쉬 함수 라이브러리인 hashlib를 import해주어야한다.

sha256() : SHA-256 해시 객체를 생성해준다.
.digest() : digest는 hash 함수를 통과한 이후의 데이터를 의미한다. 이를 활용하면 메소드에 전달된 데이터의 digest를 반환한다,
.hexidigest() : digest가 16진수 숫자만 포함하는 두 배 길이의 문자열 객체로 반환한다.
