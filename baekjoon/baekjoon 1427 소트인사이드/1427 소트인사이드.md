array = input()

for i in range(9, -1, -1): # 범위를 9부터 -1 전까지, -1을 해주며 반복해준다.
    for j in array:

### range()

range 함수의 인자가 위처럼 세개인 경우는 뭐지?  
인자가 두개인 경우에는 시작 숫자와 끝 숫자였다.   
인자가 세개은 경우에, 맨 마지막 인자는 각 수 사이의 거리를 의미한다.  
즉 위의 range(9, -1, -1)은 9부터 -1을 해주면서 9가 -1이 되기 전까지의 범위를 의미한다.
