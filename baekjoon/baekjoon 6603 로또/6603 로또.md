#### Question : [백준 6603](https://www.acmicpc.net/problem/6603)

## 최종 답안

```javascript
from itertools import combinations

while True:
    test_case = list(map(int, input().split()))
    if test_case[0] == 0:
        break
    num = test_case[0]
    lotto = test_case[1:]

    result = list(combinations(lotto, 6))

    for i in result:
        for j in i:
            print(j, end=' ')
        print()

    print()
```

## 조합(combinations)

조합이란 n개의 원소를 사용해서 순서의 관계없이 r개의 배열로 나타내는 것을 말한다. 조합은 순서가 없어서 원소의 종류가 같으면 같은 배열로 나타낸다.
itertools 모듈의 외장함수로 사용이 가능하다.
