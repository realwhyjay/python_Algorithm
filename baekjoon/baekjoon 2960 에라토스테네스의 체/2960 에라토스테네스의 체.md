#### Question : [백준 2960](https://www.acmicpc.net/problem/2960)

## 최종 답안 

```python
N, K = map(int, input().split())
count = 0
nums = [True] * (N+1)

for i in range(2, len(nums)+1):
    for j in range(i, N+1, i):
        if nums[j] == True:
            nums[j] = False
            count += 1
            if count == K:
                print(j)
                break

```

입력한 길이보다 하나 더 긴 길이의 True list를 만들어주고,  
2의 배수, 3의 배수 순으로 소거하여 소수를 남기는 방식이다.  
그렇게 소수가 아닌 수를 찾을 때마다 count를 체크하고, False로 값을 바꾸어주다가  
count의 수가 입력받은 수와 같아지는 경우에 break 하여 답을 출력한다.
