# 백준 1568 : 새

#### Date : 21/03/02

#### Lang : Python

#### Question : [백준 1568](https://www.acmicpc.net/problem/1568)

## 최종 답안

```python
N = int(input()) # 입력받을 새의 수

K = 1
count = 0

while N > 0: # 새가 남아있는 경우에만 진행!
    if N < K: # 남아있는 새의 수보다 K가 큰 경우 1로 초기화한다.
        K = 1
    else: # 일반적인 경우에는 K만큼 새를 달려보낸다.
        N -= K
        K += 1
        count += 1

print(count)

```
