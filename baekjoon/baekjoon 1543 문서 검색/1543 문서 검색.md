# 백준 1543 : 문서 검색

#### Date : 21/03/02

#### Lang : Python

#### Question : [백준 1543](https://www.acmicpc.net/problem/1543)

## 최종 답안

```python
document = input() # 문서
text = input() # 검색하려는 단어

index = 0 
count = 0

while index <= len(document): # 전체 문서크기보다 작거나 같은 경우에만 계속 반복하도록 한다.
    if document[index:index+len(text)] == text: 
      # 만일 index부터(초기 값은 0) 검색하려는 단어크기만큼의 문자가 text와 일치한다면
      # 최대 몇 번 등장하는지를 체크하기 위해 count를 1 늘려주고
        count += 1
      # 다음 검색을 위해 검색하고자 하는 단어 길이만큼 index를 증가시켜준다
        index += len(text)
    else:
      #만일 중복되는 단어를 찾지 못했다면 index를 1만큼만 증가시켜서 다시 비교한다.
        index += 1

print(count)
```
