## 최종 답안 

```python
n = int(input())
area = list(map(int, input().split()))
limit = int(input())

start = 1
end = max(area)

while start <= end:
    mid = (start+end)//2
    result = 0

    for i in area:
        if i > mid:
            result += mid
        else:
            result += i

    if result > limit:
        end = mid - 1
    else:
        start = mid + 1

print(end)

```
