## 최종 답안 

```python
num = int(input())
arr = [0]*(num+1)


for n in range(1, num+1):
    if n == 1:
        continue
		# 예를 들어 n이 6일 경우. 
    # 2로 나누었을때와 3으로 나누었을 때를 비교하여
    # 더 작은 값을 가진 경우를 리턴하도록 한다.
    value = []
    if n % 3 == 0:
        value.append(arr[n//3]+1)
    if n % 2 == 0:
        value.append(arr[n//2]+1)
    value.append(arr[n-1]+1)

    arr[n] = min(value)


print(str(arr[num]))
```

DP문제라고 해서 재귀 함수를 활용하려고 했는데,  
이를 활용하면 메모리 초과가 난다고 하여 for 문을 사용했다.

## 모범답안

```python
n = int(input())

dp = [0 for _ in range(n+1)]

for i in range(2, n+1):
    dp[i] = dp[i-1] + 1  

    if i%2 == 0 and dp[i] > dp[i//2] + 1 :
        dp[i] = dp[i//2]+1
        
    if i%3 == 0 and dp[i] > dp[i//3] + 1 :
        dp[i] = dp[i//3] + 1
        
print(dp[n])
```

사실 구조는 비슷하지만  
위의 답안으로는 //2, //3 의 경우의 수를 value 값에 append 해서 가장 작은 수를 리턴했주었는데  
해당 답안의 경우로는 배열에 append 하는 과정이 빠져있어서 가져와보았다.
