## 최종 답안

```python
num = int(input())
ans = []

for _ in range(num):
    data = list(map(int, input().split()))
    ans.append(data)

ans = sorted(ans, key=lambda x: (x[0], x[1]))

for i in ans:
    print(i[0], i[1])

```



### sorted() 함수

10814번 문제를 풀었던 것 처럼 key에 lambda식을 사용해서 정렬하도록 하였다.   
이 문제의 경우에는 첫번째 원소로 정렬을 하되, 첫번째 원소가 같은 경우에는 두번재 원소를 기준으로 정렬을 해야한다.   
그렇기에 sorted의 key를 key=lambda x: (x[0], x[1]) 로 해주었다.

```python
n = int(input())

array = []
for _ in range(n):
    x, y = map(int, input().split(' '))
    array.append((x, y))
array = sorted(array)

for i in array:
    print(i[0], i[1])

```
