# 백준 10808 : 알파벳 개수

#### Date : 21/07/20

#### Lang : Python

#### Question : [백준 10808](https://www.acmicpc.net/problem/10808)

## 최종 답안 

```python
string = list(input())
n = [0]*26

for i in range(len(string)):
    n[ord(string[i])-ord('a')] += 1

for i in n:
    print(i, end=' ')

```

10809 알파벳 찾기를 기반으로 하여 쉽게 풀이가 가능했다!
