# 백준 1158 : 요세푸스 문제

#### Date : 21/06/03

#### Lang : Python

#### Question : [백준 1158](https://www.acmicpc.net/problem/1158)

## 최종 답안 

```python
N, K = map(int, input().split())

list = [i for i in range(1, N+1)]
result = []
next = 0

for i in range(N):
    next += K-1
    if next >= len(list):
        next %= len(list)

    result.append(str(list.pop(next)))

print("<", ", ".join(result)[:], ">", sep='')

```

처음에는 단순히 K만큼의 차이를 두고 값을 처리해주면 된다고 생각했지만,   
값이 pop 된 이후에는, 남아있는 값 중에서 K 만큼 차이가 있는 값을 출력해주어야만했다.

뿐만 아니라 join을 사용하기 위해선, 문자열 리스트여야 한다.   
근데 이 사실을 까먹고 str 형변환 처리를 해주지 않아 오류가 생겼었다 ㅠ.
