## 1차 답안 

```python
str = list(input())
curser = len(str)

for _ in range(int(input())):
    inst = list(input().split())
    if inst[0] == 'P':
        str.insert(curser+1, inst[1])

    elif inst[0] == 'L':
        if curser > 0:
            curser -= 1

    elif inst[0] == 'D':
        if curser != len(str)+1:
            curser += 1

    else:
        if curser >= 0:
            if curser == len(str):
                str.pop(-1)
                curser -= 1
            else:
                str.pop(curser)
                curser -= 1
for i in str:
    print(i, end='')

```

이렇게 하면 답은 제대로 출력하지만, 시간초과가 나온다.  
insert를 해줬기 때문인데, 시간 복잡도가 O(n)이 나와 시간 초과가 나는것!   
이걸 해결하기 위해선 단순히 append와 pop만을 사용해야한다고한다.  

## 모범 답안

```python
import sys

str1 = list(sys.stdin.readline().strip())
str2 = []

for _ in range(int(input())):
    inst = list(sys.stdin.readline().strip())
    if inst[0] == 'P':
        str1.append(inst[2])
    elif inst[0] == 'L' and str1 != []:
        str2.append(str1.pop())
    elif inst[0] == 'D' and str2 != []:
        str1.append(str2.pop())
    elif inst[0] == 'B' and str1 != []:
        str1.pop()

print("".join(str1 + list(reversed(str2))))

```

커서의 위치를 확인해서 insert를 해주었던 나의 코드와 달리,    
stack 두개를 만들어서 커서의 이동에 따라 각 스택에 해당 문자를 옮기는 식으로 구현을 하였다.  

커서가 왼쪽으로 이동하면, srt1에 있는 원소를 str2로 넘기고, 오른쪽으로 이동하면 str2에 있는걸 str1으로 옮기는 식. 
이렇게 구현을 한다면 시간복잡도가 O(1)로 시간 초과를 피할 수가 있다.
