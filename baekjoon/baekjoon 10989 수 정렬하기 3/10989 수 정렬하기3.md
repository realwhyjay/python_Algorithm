## 1차 답안

```python
num = int(input())

ans = []
for _ in range(num):
    ans.append(int(input()))

ans.sort()

for i in ans:
    print(i)

```

해당 답안은 원하는 값을 출력하긴 하지만, 백준에 답안을 제출했을 때 메모리 초과가 나왔다.  
찾아보니 for문에서 append를 해주면 메모리를 매번 재할당 하기 때문에, 메모리 효율이 좋지 않다고 한다.

## 최종 답안

```python
import sys

# 시간 초과를 방지하기 위해, input() 대신 sys를 import하여 readline()을 사용했다
num = int(sys.stdin.readline())
'''
이전에는 빈 list를 만들어주고 매번 qppend 해줬지만 이번에는 0으로 가득찬 list를 만들어두고
입력받은 수를 index로 하는 원소에 1씩 더해주도록 했다.
이후 차례대로, 입력된 수 만큼 출력하는 방식이다.
'''
ans = [0] * 10001
# 입력값이 10000까지 들어올 수 있으니 10001 만큼 list를 채워준다. (index는 0부터 시작하니까)
for _ in range(num):
    ans[int(sys.stdin.readline())] += 1

for i in range(10001):
    if ans[i] != 0:
        for j in range(ans[i]):
            print(i)

```


파이썬은 내부적으로 연산과 메모리를 관리하기 때문에, 
파이썬에 내장되어있는 함수를 사용할 수록 메모리를 효율적으로 관리할 수 있다고 한다.
