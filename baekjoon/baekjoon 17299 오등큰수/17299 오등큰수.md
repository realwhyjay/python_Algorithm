# 백준 17299 : 오등큰수

#### Date : 21/06/27

#### Lang : Python

#### Question : [백준 17299](https://www.acmicpc.net/problem/17299)

## 최종 답안 

```python
import sys
from collections import Counter

input = sys.stdin.readline

N = int(input())
A = list(map(int, input().split()))
cnt = Counter(A)

stack = []

# 오큰수를 못찾은 값은 -1로 출력해주기 위해 -1로 result를 채워준다
result = [-1] * N
A = [[cnt[num], int(num)] for num in A]

stack.append(0)

i = 1

# 스택에 값이 있고, i가 N보다 작을 경우에
while stack and i < N:
    # 스택에 값이 있고, 스택의 가장 위에 있는 인덱스에 해당하는 값의 count 수보다 인덱스 i에 해당하는 count 수가 더 크면
    while stack and A[stack[-1]][0] < A[i][0]:
        result[stack[-1]] = A[i][1]  # result 리스트의 해당 인덱스에 값을 저장한다
        stack.pop()  # 오등큰수를 찾은 인덱스는 pop을 해준다.

    stack.append(i)
    i += 1

for i in result:
    print(i, end=' ')

```

해당 문제는 17298 오큰수 문제에 간단한 조건이 추가된 문제이다.  
로직은 똑같지만, count를 해주는 과정을 제대로 떠올리지 못해 구글링을 했다.  

### **from** collections **import** Counter

collections 모듈의 Counter 클래스를 사용하면,   
해당 리스트 또는 문자열에 포함된 원소들의 개수를 카운팅 할 수 있다.  

해당 코드의 cnt를 출력해보면 `Counter({1: 3, 2: 2, 3: 1, 4: 1})`와 같이,  
dictionary 형식으로 출력되는걸 확인할 수 있다.
