#### Date : 21/06/02

#### Lang : Python

#### Question : [백준 9093](https://www.acmicpc.net/problem/9093)

## 최종 답안 

```python
T = int(input())

for _ in range(T):
    word = list(map(list, input().split()))
    for i in word:
        for j in reversed(i):
            print(j, end='')
        print(' ', end='')
    print()

```

시간이 너무 오래걸리길래 답안을 찾아보았는데.. 

## 모범답안

```python
n = int(input())

for _ in range(n):
    arr = list(map(list,input().split()))
    for i in arr:
        print("".join(i[::-1]),end=" ")
```

이렇게 간단한 방법이 있었는데, 내가 너무 어렵게 생각했던듯 하다.
[::-1] 슬라이싱을 활용해 뒤집고 join 함수를 활용하여 합쳐주어 원하는 값을 출력한다.
