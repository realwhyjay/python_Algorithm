# 백준 1978 : 소수 찾기

#### Date : 21/01/30

#### Lang : Python

#### Question : [백준 1978](https://www.acmicpc.net/problem/1978)


## 최종 답안

```python
# 주요 값들을 입력받는다
case_num = int(input())
num_list = list(map(int, input().split()))
ans = 0

# 입력 받은 케이스의 수와 리스트의 요소 갯수가 같을 때만 진행하도록 조건을 걸어준다
if case_num == len(num_list):
    for i in num_list:
        count = 0
        # 소수는 자기 자신과 1 외에 나눠지는 수가 없는 것이니까, 정확히 나눠질 때마다 카운트를 올려주고
        # 그 수가 2가 될때에만 출력할 ans 값을 1씩 올린다
        for j in range(1, i+1):
            if i % j == 0:
                count += 1
        if count == 2:
            ans += 1

print(ans)
```
